## Description

MoltyDEX is the first DEX aggregator built specifically for x402 payments. It automatically swaps tokens when making x402 payments, making it seamless for AI agents to pay for APIs.

**Key Features:**
- Automatic token swapping for x402 payments
- Best prices across all Solana DEXes (Jupiter routing)
- Simple REST API and Python SDK
- Client-side signing (secure)
- Low fees (0.1%)

## Use Cases

1. **AI Agents** - Automatically swap tokens when receiving 402 Payment Required responses
2. **Traders** - Get best prices across all Solana DEXes
3. **Developers** - Integrate x402 payments into your applications

## Technical Details

- **Frontend:** Next.js
- **Backend:** Express.js (Node.js)
- **Aggregator:** Jupiter
- **Network:** Solana Mainnet
- **Signing:** Client-side (keys never leave user's system)
