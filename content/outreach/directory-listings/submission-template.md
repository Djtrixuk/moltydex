## Detailed Description (Long Form)

MoltyDEX is a fast, cheap DEX aggregator for Solana that works for both humans and AI agents. For humans: simple web interface for instant token swaps. For agents: automatic x402 payment handling. Unlike other aggregators, MoltyDEX offers the lowest fees (0.1%) and best prices via Jupiter routing.

**Key Features:**
- **Automatic Token Swapping:** Detects 402 Payment Required responses and automatically swaps tokens when needed
- **x402 Native:** Complete x402 protocol support with automatic payment handling
- **Best Prices:** Routes through Jupiter aggregator to find optimal prices across all Solana DEXes
- **Client-Side Signing:** Maximum security - private keys never leave your system
- **Agent-Friendly API:** Built specifically for programmatic use cases
- **Low Fees:** 0.1% per swap, no subscriptions, no hidden fees

**Use Cases:**
- **For Traders:** Fast, cheap token swaps with best prices
- **For Agents:** Automatic x402 payment handling
- Premium API access for AI agents
- Automated data collection from multiple paid APIs
- Pay-per-use AI services
- x402 protocol integration

**Technology:**
- Built on Solana blockchain
- Uses Jupiter aggregator for routing
- TypeScript/JavaScript SDK
- Python SDK
- REST API

- Open source and auditable
- Client-side signing only
