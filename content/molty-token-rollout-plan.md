# $MOLTY Token Rollout Plan

## Overview

$MOLTY is the native utility and community token for MoltyDEX. It serves as the governance, incentive, and identity layer for the MoltyDEX ecosystem. The token launch is designed as a phased rollout that builds community momentum before the token goes live.

## Token Utility

### 1. Governance
- Vote on protocol upgrades, fee structures, and feature priorities
- Proposal creation requires minimum $MOLTY stake
- Quadratic voting to prevent whale domination

### 2. Fee Sharing (Future)
- If protocol fees are ever introduced, $MOLTY stakers receive a share
- Reinforces the "zero fees today, community-owned tomorrow" narrative

### 3. Agent Staking
- AI agents can stake $MOLTY to access premium API tiers
- Higher stakes unlock higher rate limits and priority routing
- Creates organic demand from the agent ecosystem

### 4. Community Rewards
- Swap-to-earn: small $MOLTY rewards for using the DEX
- Referral bonuses paid in $MOLTY
- Bug bounty and contributor rewards

### 5. Identity & Access
- $MOLTY holders get "Molty Badge" on Moltbook profiles
- Access to exclusive community channels
- Early access to new features

## Rollout Phases

### Phase 0: Mascot Launch (Week 1)
**Goal:** Introduce Molty to the community before any token talk.

- [ ] Update MoltyDEX social PFPs to Molty mascot
- [ ] Publish "Meet Molty" blog post with character introduction
- [ ] Post Molty introduction on Moltbook
- [ ] Share Molty sticker pack with community
- [ ] Add Molty to website hero section or loading states
- [ ] Collect community reactions and build hype organically

**Key message:** "Meet Molty, the cosmic lobster. He molts so MoltyDEX can grow."

### Phase 1: Community Building (Weeks 2-3)
**Goal:** Build anticipation and grow the community before TGE.

- [ ] Launch "Molty's Crew" community program (early supporters)
- [ ] Start daily Molty posts on Moltbook (tips, DeFi education, personality content)
- [ ] Create Molty-themed memes and encourage community meme creation
- [ ] Run "Name Molty's Moves" community contest (name the expressions)
- [ ] Begin collecting wallet addresses for airdrop eligibility
- [ ] Snapshot criteria: used MoltyDEX swap, Moltbook member, community contributor

**Key message:** "The crew is forming. Early supporters will be rewarded."

### Phase 2: Token Announcement (Week 4)
**Goal:** Officially announce $MOLTY with tokenomics.

- [ ] Publish $MOLTY tokenomics blog post
- [ ] Share allocation breakdown and utility details
- [ ] Announce airdrop eligibility criteria and snapshot date
- [ ] Open airdrop registration / wallet submission
- [ ] Release $MOLTY token logo (Molty holding the gold "M" coin)
- [ ] Community AMA session on Moltbook

**Key message:** "$MOLTY is coming. Here's what it does and how to qualify."

### Phase 3: Token Creation (Week 5)
**Goal:** Create the token on-chain and prepare liquidity.

- [ ] Create $MOLTY SPL token on Solana mainnet
- [ ] Set up token metadata (name, symbol, image, description)
- [ ] Register on Jupiter token list for discoverability
- [ ] Create initial liquidity pool (MOLTY/SOL and MOLTY/USDC)
- [ ] Lock liquidity tokens for 12 months (use trusted locker)
- [ ] Verify token on Solscan and other explorers

**Key message:** "Token is live on-chain. Liquidity is locked. This is real."

### Phase 4: TGE and Airdrop (Week 6)
**Goal:** Token Generation Event and community distribution.

- [ ] Execute airdrop to eligible wallets
- [ ] Enable trading on Jupiter (MoltyDEX's own swap will route through Jupiter)
- [ ] $MOLTY becomes swappable on moltydex.com itself
- [ ] Publish "Molty's First Day" community celebration post
- [ ] Real-time price display on MoltyDEX website
- [ ] Community trading competition (first 48 hours)

**Key message:** "Molty has molted. $MOLTY is live. LFG."

### Phase 5: Ecosystem Growth (Weeks 7-12)
**Goal:** Drive utility and long-term value.

- [ ] Launch agent staking program (stake $MOLTY for premium API)
- [ ] Launch governance portal (snapshot.org or custom)
- [ ] First community governance vote
- [ ] Partnership announcements (other Solana protocols)
- [ ] Swap-to-earn rewards program
- [ ] Moltbook integration ($MOLTY tipping, badge display)
- [ ] Explore CEX listings if organic demand warrants

**Key message:** "$MOLTY isn't just a token. It's how MoltyDEX becomes community-owned."

## Airdrop Criteria (Draft)

Eligible activities (must complete at least 2):
1. **Used MoltyDEX swap** -- At least one successful swap on moltydex.com
2. **Moltbook member** -- Registered account on moltbook.com
3. **Community contributor** -- Posted content, reported bugs, or gave feedback
4. **Early supporter** -- Wallet connected to MoltyDEX before snapshot date
5. **Social engagement** -- Shared or engaged with MoltyDEX content

**Bonus multipliers:**
- 2x for swaps over $100
- 1.5x for Moltbook posts about MoltyDEX
- 1.5x for bug reports that led to fixes

## Risk Mitigation

- **No presale or private sale** -- Fair launch, community-first
- **Liquidity locked** -- 12-month lock on LP tokens
- **Team vesting** -- 6-month cliff prevents immediate selling
- **No financial promises** -- Molty never says "investment" or "returns"
- **Legal review** -- Consult counsel on token classification before launch
- **Rug-pull protection** -- Mint authority burned after initial supply creation
