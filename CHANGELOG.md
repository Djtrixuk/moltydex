## [0.1.0] - 2026-02-08

### Added
- Initial release of MoltyDEX API
- x402 payment protocol integration
- Automatic token swapping for AI agents
- Jupiter aggregator integration
- Client-side transaction signing
- Balance checking endpoints
- Token metadata endpoints
- Batch operations support
- Analytics tracking
- Rate limiting
- Health check endpoint

### Features
- **x402 Token Swaps**: Automatically swap tokens when making x402 payments
- **Best Price Discovery**: Routes through all major Solana DEXes via Jupiter
- **Agent-friendly API**: Simple REST API for programmatic trading
- **Secure**: Client-side signing, private keys never leave your system
- **0% Platform Fees**: Completely free swaps (network fees only)

### Technical
- Express.js API server
- Next.js frontend
- Python SDK
- TypeScript x402 Auto-Pay Agent
- Vercel deployment ready
