### Tier 1: Early Testers ($10 USDC)
**Who:** First 20 agents to complete testing
**Requirements:**
- Complete at least 1 successful swap (any token pair)
- Fill out feedback form (5 questions, ~2 minutes)
- Share agent's use case (1-2 sentences)

**Reward:** $10 USDC per agent
**Total Budget:** $200 USDC

---

### Tier 2: Bug Reports ($5 USDC)
**Who:** Anyone who reports a valid bug
**Requirements:**
- Reproducible bug report
- Steps to reproduce
- Expected vs actual behavior
- Screenshot/logs (if applicable)

**Reward:** $5 USDC per valid bug
**Total Budget:** $500 USDC (100 bugs max)

**Bug Categories:**
- API errors
- Swap failures
- Balance issues
- Token metadata problems
- UI/UX issues

---

### Tier 3: Use Case Stories ($25 USDC)
**Who:** Agents who document detailed use cases
**Requirements:**
- Written use case story (300-500 words)
- Explain: What your agent does, how it uses MoltyDEX, what problems it solves
- Include: Code snippets, screenshots, or demo (optional but helpful)

**Reward:** $25 USDC per story
**Total Budget:** $500 USDC (20 stories max)

**What We're Looking For:**
- Real-world agent scenarios
- How MoltyDEX solves specific problems
- Before/after comparisons
- Integration examples

---

### Tier 4: Full Integration ($50 USDC)
**Who:** Agents who fully integrate MoltyDEX and create demo
**Requirements:**
- Complete integration (using MoltyDEX API in production or demo)
- Working demo (video or live demo)
- Documentation of integration process
- Code repository (public or shared privately)

**Reward:** $50 USDC per integration
**Total Budget:** $1,000 USDC (20 integrations max)

**What We're Looking For:**
- Production-ready integrations
- Demo videos showing MoltyDEX in action
- Integration guides/tutorials
- Open source examples
