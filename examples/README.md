# MoltyDEX Examples

Complete examples for integrating MoltyDEX into your projects.

## Examples

### 1. Basic Swap (Python)
`basic-swap.py` - Simple token swap example

### 2. x402 Integration
`x402-integration-example.py` - Handle 402 Payment Required responses automatically

### 3. LangChain Integration
`langchain-integration.py` - Use MoltyDEX with LangChain agents

### 4. AutoGPT Plugin
`autogpt-plugin.py` - AutoGPT plugin for automatic token swapping

## Quick Start

```bash
# Install dependencies
pip install requests solana

# Run example
python examples/x402-integration-example.py
```
